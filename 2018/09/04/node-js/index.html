    什么是node.js

    node.js是一个构建在浏览器v8引擎上的js运行环境

    node.js 使用了事件驱动 非阻塞I/O模型 这些都使它轻量 好用

    node.js的包生态(npm) 是世界上最大的开源库生态系统

    安装node.js

    node.js可以在任何操作系统上安装 底层是Chrome V8引擎

    官网 www.nodejs.org下载
    找对应的操作系统的版本就行了
    目前有Windows Linux mac 记住安装路径不能有中文路径
